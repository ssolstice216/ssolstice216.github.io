AuraRAW Help & FAQ
Quick answers to common questions about camera features, storage, and compatibility.
How can I save to my external SD card?
This depends on your Android version:
  • Android 5.0+ – Enable Settings → Storage → Storage Access Framework to allow saving to external SD cards. If only “Recent” appears, open the top-right menu and enable showing drives.
  • Android 6.0+ – Some devices support Adoptable Storage (SD as internal storage). If not supported, use the Storage Access Framework method.
Can I disable the shutter sound?
Switch Settings → Audio to Camera2 API to unlock shutter sound control. If the option is missing, your device’s camera API does not allow third-party apps to disable it.
